# Methodology Prompt

## 使用方法

将此 Prompt 发送给 Paper Writer Agent，替换 `{变量}` 为你的实际内容。

---

请为以下论文撰写研究方法章节：

**研究问题**：{研究问题}
**研究范式**：{定量/定性/混合}
**研究方法**：{具体方法，如问卷调查、实验、案例研究等}
**研究对象**：{研究对象/样本}
**数据来源**：{数据来源}
**分析工具**：{统计软件/分析方法}

### 输出结构（以定量研究为例）

```
3.1 研究框架与假设
    3.1.1 研究模型
    3.1.2 研究假设

3.2 研究设计
    3.2.1 研究范式选择及理由
    3.2.2 变量定义与操作化
        - 自变量
        - 因变量
        - 控制变量
        - 中介/调节变量（如有）

3.3 数据收集
    3.3.1 样本选择与抽样方法
    3.3.2 样本量确定
    3.3.3 问卷/数据收集工具
    3.3.4 数据收集过程

3.4 数据分析方法
    3.4.1 描述性统计
    3.4.2 信效度检验
    3.4.3 假设检验方法

3.5 研究伦理
```

### 写作规范

1. 每个方法选择都要说明理由
2. 变量操作化要给出明确定义和测量方式
3. 样本选择要说明抽样方法和依据
4. 数据分析方法要与研究假设对应
5. 详细到他人可以复现的程度
6. 字数建议：3000-6000字
