# 快速开始

## 安装

```bash
git clone https://github.com/WuSuBuDuoMing/paper-writer.git
cd paper-writer
cp -r .claude/ /path/to/your/paper-project/.claude/
```

## 使用方法

在你的论文项目目录中启动 Claude Code：

```bash
claude
```

然后使用内置命令：

### 生成大纲
```
/outline "你的研究主题"
```

### 文献综述
```
/literature "研究方向关键词"
```

### 设计研究方法
```
/method "研究问题"
```

### 撰写摘要
```
/write abstract
```

### 润色文本
```
/polish "需要润色的文本"
```

### 自审论文
```
/review
```
