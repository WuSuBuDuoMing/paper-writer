# Discussion Prompt

## 使用方法

将此 Prompt 发送给 Paper Writer Agent，替换 `{变量}` 为你的实际内容。

---

请为以下论文撰写讨论与结论章节：

**研究问题**：{研究问题}
**主要发现**：{列出所有主要发现}
**与已有研究的对比**：{与文献综述中已有研究的异同}
**理论贡献**：{对理论的贡献}
**实践启示**：{对实践的建议}
**目标字数**：{字数要求，默认 3000-5000 字}

### 输出结构

```
5.1 主要发现讨论
    5.1.1 [发现1] 的讨论
    5.1.2 [发现2] 的讨论
    ...

5.2 理论贡献
    5.2.1 对 [理论X] 的贡献
    5.2.2 对 [理论Y] 的补充

5.3 实践启示
    5.3.1 对 [领域/行业] 的建议
    5.3.2 对 [管理者/政策制定者] 的建议

5.4 研究局限性
    - 局限性1及未来研究方向
    - 局限性2及未来研究方向
    - 局限性3及未来研究方向

5.5 结论
    - 研究总结（回应研究问题）
    - 未来研究展望
```

### 写作规范

1. 讨论要回应引言中提出的研究问题
2. 与文献综述中的已有研究进行对比讨论
3. 解释意外发现的可能原因
4. 理论贡献要具体，不要泛泛而谈
5. 实践启示要可操作
6. 局限性要诚恳，但也要说明如何缓解
7. 字数建议：3000-5000字
8. 讨论部分可适当使用 hedging 语言（可能、推测、表明）

### 示例输入

```
研究问题：AI 辅助编程教学是否显著提升学生的编程能力？

主要发现：
1. 实验组编程成绩显著高于对照组（d=0.59, 中等效应）
2. 学习动机在 AI 辅助教学与成绩间起部分中介作用
3. 低基础学生受益更大（调节效应显著）

与已有研究的对比：
- 与 Smith et al. (2024) 结论一致，但本研究效应量更大
- 与 Li & Wang (2023) 关于动机中介的结论互补

理论贡献：拓展了 TAM 在编程教育场景的适用性
实践启示：建议高校引入 AI 辅助教学工具，优先用于基础薄弱学生
```
