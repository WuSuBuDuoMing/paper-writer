## 多智能体协作流程

### 全流程协作

一个完整的论文写作流程通常涉及多个 Agent 协作：

```
1. Paper Writer Agent → 选题、大纲生成
2. Literature Reviewer Agent → 文献检索与综述
3. Methodology Designer Agent → 研究设计
4. Data Analyst Agent → 数据分析与结果撰写
5. Academic Editor Agent → 语言润色
6. Citation Manager Agent → 引用格式化
7. Paper Writer Agent → 最终质量审查
```

### 快速切换 Agent

在 Claude Code 会话中，你可以按需切换 Agent：

```bash
# 先用主 Agent 生成大纲
claude "/outline '基于深度学习的遥感图像分类方法研究'"

# 切换到文献综述 Agent
claude "/literature 'deep learning remote sensing classification'"

# 切换到方法论 Agent
claude "/method 'CNN 实验设计 监督学习'"

# 切换到数据分析师
claude "/write results"

# 最后润色和引用
claude "/polish [文本段落]"
claude "/cite [参考文献列表]"
```

### 中间审查点

在每个阶段之间进行审查：

```
/outline → 审查大纲 → /literature → 审查综述 → /method → 审查方法
→ /write abstract → 审查摘要 → /write results → 审查结果 → /review
```

## 跨语言论文写作

### 中文论文 → 英文论文

1. 先用中文完成全文写作
2. 使用 Academic Editor Agent 进行翻译和润色

```
请将以下中文学术段落翻译为英文学术论文风格：
「本研究采用准实验设计，以300名大学生为研究对象，考察了AI教学工具对学习效果的影响。结果表明，实验组的学业成绩显著高于对照组（p < 0.01）。」
```

### 英文论文 → 中文论文

```
请将以下英文学术段落翻译为中文学术论文风格（非逐字翻译，需符合中文论文写作习惯）：
"This study employed a quasi-experimental design with 300 undergraduate students
to investigate the impact of AI teaching tools on learning outcomes. The results
demonstrated that the experimental group significantly outperformed the control
group (p < .001)."
```

### 中英摘要对照撰写

```
请同时撰写以下论文的中英文摘要（摘要内容对应但非逐字翻译）：

研究主题：大语言模型在高等教育教学中的应用效果研究
研究方法：准实验设计，前后测对照组，N=300
主要发现：LLM辅助教学组成绩提升15.2%，学习动机提升23.6%
结论：大语言模型可有效辅助高等教育教学，建议推广
```

## 期刊投稿定制

### 针对特定期刊格式

每个期刊有独特的格式要求。请在指令中明确目标期刊：

```
请按《教育研究》期刊的投稿要求，调整论文格式：
- 严格使用 GB/T 7714-2015 引用格式
- 中文摘要 300-500 字，英文摘要 200 词以内
- 关键词 3-8 个
- 图表随文排列
- 参考文献不少于 30 篇
```

```
Please format this paper for IEEE Transactions submission:
- IEEE reference style with numbered citations [1], [2]
- Two-column format
- Maximum 8 pages (including references)
- IEEE section headings (I. INTRODUCTION, II. RELATED WORK, etc.)
- Mathematical equations in numbered format
```

### 不同学科领域的投稿适配

| 学科 | 推荐格式 | 推荐 Agent 组合 |
|------|---------|----------------|
| 教育学 | GB/T 7714 | 文献综述 + 方法论 + 数据分析 |
| 计算机科学 | IEEE | 方法论 + 数据分析 + 引用管理 |
| 心理学 | APA 7th | 方法论 + 数据分析 + 学术编辑 |
| 工程学 | IEEE | 方法论 + 数据分析 + 引用管理 |
| 管理学 | GB/T 7714 或 APA | 全流程 |

## 学位论文写作

### 硕士论文特殊要求

```
请按照硕士论文规范撰写第3章研究方法部分：
- 需包含完整的研究伦理声明
- 变量操作化要详细到可复现
- 问卷量表需附完整题项
- 样本量说明和统计功效分析
- 本文字数约 3000-5000 字
```

### 博士论文特殊要求

```
请按照博士论文规范撰写第2章文献综述部分：
- 需包含系统性综述方法论说明（如 PRISMA 流程图）
- 文献覆盖近 10 年的核心文献
- 综述需达到批判性分析深度
- 明确指出研究空白并论证本研究的创新点
- 本章字数约 15000-20000 字
```

### 论文结构规范

| 部分 | 硕士论文建议字数 | 博士论文建议字数 |
|------|:--------------:|:--------------:|
| 摘要 | 500-1000字 | 800-1500字 |
| 引言 | 3000-5000 | 8000-15000 |
| 文献综述 | 5000-8000 | 15000-25000 |
| 研究方法 | 3000-5000 | 8000-15000 |
| 结果 | 3000-5000 | 8000-15000 |
| 讨论与结论 | 3000-5000 | 8000-15000 |

## 文献综述专题

### PRISMA 系统性综述流程

```
请按 PRISMA 2020 标准帮我完成系统性文献综述：
1. 制定检索策略（至少3个数据库）
2. 设计纳入/排除标准表
3. 绘制 PRISMA 流程图描述
4. 使用 PICOS 框架明确研究问题
5. 对纳入文献进行质量评估
```

### 文献矩阵表

```
请帮我生成一个文献矩阵表，包含以下列：
| 作者 | 年份 | 研究目的 | 研究方法 | 样本 | 核心发现 | 局限性 |

涵盖以下文献：
1. Wang et al. (2024): AI教学工具效果研究
2. Li & Chen (2023): 个性化学习系统评估
3. Zhang et al. (2024): 大语言模型教育应用
```

## 数据可视化与结果呈现

### 三线表设计

```
请帮我设计以下数据的三线表（APA格式）：
变量：感知有用性、感知易用性、社会影响、使用意愿
分组：实验组、对照组
统计量：均值、标准差、t值、p值
```

### 图表描述撰写

```
请为以下图表撰写描述文字：

图1：研究模型图
- 包含自变量（感知有用性、感知易用性）
- 中介变量（态度）
- 因变量（使用意愿）
- 控制变量（年龄、性别、技术经验）
```

## 批量引用管理

### 批量格式转换

```
请将以下10条文献信息统一转换为 IEEE 格式：
1. (王明, 2024) 教育研究, 45(3), 12-20
2. Chen W, Nature Education, 2023, 15(2)
3. ...
（粘贴完整文献列表）
```

### 引用一致性全面检查

```
请对我的论文进行全面的引用一致性检查：
1. 文内引用 vs 参考文献列表：有无遗漏或多出
2. 引用格式：是否统一
3. 作者姓名：中英文格式是否一致
4. 年份：是否有缺失
5. DOI：是否正确
6. 排序：是否按要求排列
```

## 多版本修订工作流

### 逐轮修订策略

```
第一轮修订：结构和内容
- 检查论文结构是否合理
- 检查论证逻辑是否完整
- 检查各章节比例是否恰当

第二轮修订：学术语言
- 润色学术表达
- 消除口语化表述
- 检查时态、语态一致性

第三轮修订：格式和引用
- 统一引用格式
- 检查图表编号
- 核对参考文献列表

第四轮修订：最终审查
- 通读全文检查流畅性
- 检查拼写和标点
- 确认页码和目录
```

### Reviewer 反馈修改

```
请根据以下审稿人意见修改论文：

审稿人1意见：
1. 样本量偏小，需补充统计功效分析
2. 请增加与 Smith (2023) 研究的对比讨论

审稿人2意见：
1. 方法论部分需更详细描述数据分析步骤
2. 建议增加局限性讨论

请逐条回应并修改相应章节。
```
