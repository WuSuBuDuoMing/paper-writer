# GB/T 7714-2015 Style Guide

## 论文结构（中文学术论文）

```
摘要
关键词
1 引言
2 文献综述
3 研究方法
4 结果与分析
5 讨论
6 结论
参考文献
致谢
附录
```

## 格式要求

### 页面设置
- 页边距：上下2.54cm，左右3.17cm
- 字体：正文宋体小四号，标题黑体
- 行距：1.5倍行距
- 段落首行缩进：2字符

### 标题层级
```
1 一级标题（黑体，三号，居中）
1.1 二级标题（黑体，四号，顶格）
1.1.1 三级标题（黑体，小四号，顶格）
```

## 文内引用格式

### 顺序编码制
```
...如文献[1]所述...
...多项研究[1-3]表明...
...根据文献[1,3,5]...
```

### 著者-出版年制
```
...如王某某(2024)所述...
...研究表明[1-3]...
```

## 参考文献格式

### 期刊文章[J]
```
[1] 作者. 题名[J]. 刊名, 出版年, 卷(期): 起止页码.
[2] 张三, 李四. 人工智能在教育中的应用研究[J]. 教育研究, 2024, 45(3): 12-20.
```

### 书籍[M]
```
[3] 作者. 书名[M]. 版次. 出版地: 出版者, 出版年: 起止页码.
[4] 王五. 机器学习导论[M]. 3版. 北京: 清华大学出版社, 2024: 56-78.
```

### 学位论文[D]
```
[5] 作者. 题名[D]. 保存地: 保存单位, 年份.
[6] 李某某. 深度学习在自然语言处理中的应用研究[D]. 北京: 北京大学, 2024.
```

### 会议论文[C]
```
[7] 作者. 题名[C]//会议录名. 出版地: 出版者, 出版年: 起止页码.
```

### 电子文献[EB/OL]
```
[8] 作者. 题名[EB/OL]. (发表日期)[引用日期]. 网址.
[9] OpenAI. GPT-4 Technical Report[EB/OL]. (2023-03-15)[2024-06-01]. https://arxiv.org/xxxxx.
```

### 作者规则
- 1-3人：全部列出，逗号分隔
- 3人以上：前3人后加"等"或"et al."
- 外文作者：姓在前名在后，名缩写
