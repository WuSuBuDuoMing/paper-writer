# 期刊论文写作示例

本示例展示使用 Paper Writer 完成一篇中文学术期刊论文的完整流程。

## 场景

研究方向：人工智能在教育领域的应用
目标期刊：《教育研究》
论文类型：实证研究

## 步骤 1：生成大纲

```
/outline "人工智能个性化学习系统对大学生学习效果的影响研究"
```

Agent 会生成类似以下大纲：

```
1 引言
  1.1 研究背景
  1.2 研究问题与目标
  1.3 研究意义
  1.4 研究内容与方法
  1.5 论文结构安排

2 文献综述
  2.1 个性化学习的理论基础
  2.2 人工智能在教育中的应用现状
  2.3 学习效果评估方法
  2.4 研究现状总结与未来方向

3 研究方法
  3.1 研究框架与假设
  3.2 研究设计
  3.3 数据收集
  3.4 数据分析方法

4 结果与分析
  4.1 描述性统计
  4.2 假设检验
  4.3 稳健性检验

5 讨论与结论
  5.1 主要发现讨论
  5.2 理论贡献
  5.3 实践启示
  5.4 研究局限性
  5.5 结论
```

## 步骤 2：文献综述

```
/literature "个性化学习 人工智能 教育技术"
```

Agent 会：
1. 制定检索策略（关键词、数据库、时间范围）
2. 按主题分类整理文献
3. 撰写系统性综述
4. 指出研究缺口

## 步骤 3：设计研究方法

```
/method "准实验设计 前后测对照组"
```

Agent 会输出完整的方法论章节，包括：
- 研究框架和假设
- 变量定义与操作化
- 样本选择和抽样方法
- 数据收集工具
- 分析方法

## 步骤 4：撰写各章节

```
/write abstract
/write introduction
/write results
/write discussion
```

## 步骤 5：润色与审查

```
/polish "本文研究了人工智能个性化学习系统对大学生学习效果的影响。通过对300名大学生进行准实验研究，结果表明..."
/review
```

## 步骤 6：格式化

```
/format GB/T 7714
```

## Prompt 模板直接使用示例

### 使用摘要模板

将 `prompts/sections/abstract.md` 中的 `{变量}` 替换为实际内容：

```
请为以下论文撰写中英文摘要：

研究主题：人工智能个性化学习系统对大学生学习效果的影响
研究问题：AI个性化学习系统如何影响大学生的学业成绩和学习动机？
研究方法：准实验设计，300名大学生，前后测对照组
主要发现：实验组学业成绩提升15.2%，学习动机提升23.6%
结论：AI个性化学习系统能显著提升学习效果，建议高校推广
```

### 使用引言模板

```
请为以下论文撰写第一章：绪论/Introduction。

研究主题：AI个性化学习系统对学习效果的影响
学科领域：教育技术学
研究背景概述：随着AI技术发展，个性化学习成为教育改革方向
核心研究问题：AI个性化学习系统如何影响学习效果？
预期贡献：构建AI个性化学习效果评估框架
```

### 使用 GB/T 7714 格式

```
请按 GB/T 7714-2015 格式生成以下文献的引用：

作者：张三, 李四
题名：人工智能在教育中的应用研究
刊名：教育研究
年份：2024
卷：45
期：3
页码：12-20
```

输出：`[1] 张三, 李四. 人工智能在教育中的应用研究[J]. 教育研究, 2024, 45(3): 12-20.`
