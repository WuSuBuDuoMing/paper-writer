# Abstract Prompt

## 使用方法

将此 Prompt 发送给 Paper Writer Agent，替换 `{变量}` 为你的实际内容。

---

请为以下论文撰写中英文摘要：

**研究主题**：{研究主题}
**研究问题**：{研究问题}
**研究方法**：{研究方法}
**主要发现**：{主要发现}
**结论**：{结论}

### 输出要求

**中文摘要**（300-500字）：
- 背景与目的（1-2句）
- 方法（1-2句）
- 结果（2-3句）
- 结论（1-2句）
- 关键词（3-5个）

**English Abstract** (150-300 words):
- Background & Objective (1-2 sentences)
- Methods (1-2 sentences)
- Results (2-3 sentences)
- Conclusion (1-2 sentences)
- Keywords (3-5)

### 摘要写作规范

1. 使用第三人称或无人称句式
2. 避免引用参考文献
3. 避免使用缩写（首次出现时）
4. 结果部分应包含具体数据
5. 语言简洁，每句话信息密度高
6. 中英文摘要内容应对应但非逐字翻译
