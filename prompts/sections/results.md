# Results Prompt

## 使用方法

将此 Prompt 发送给 Paper Writer Agent，替换 `{变量}` 为你的实际内容。

---

请为以下论文撰写结果/数据分析章节：

**研究假设/问题**：{列出所有假设或研究问题}
**分析方法**：{使用的统计分析方法}
**数据结果**：{粘贴统计分析输出结果}
**显著性水平**：{如 p < 0.05}

### 输出结构

```
4.1 描述性统计
    4.1.1 样本特征
    4.1.2 变量描述性统计
    4.1.3 相关性分析

4.2 信效度检验
    4.2.1 信度分析
    4.2.2 效度分析

4.3 假设检验
    4.3.1 [假设1] 的检验结果
    4.3.2 [假设2] 的检验结果
    ...

4.4 稳健性检验
    4.4.1 替代模型验证
    4.4.2 内生性处理（如适用）

4.5 结果汇总
    - 表格汇总所有假设检验结果
```

### 写作规范

1. 先描述样本特征，再呈现分析结果
2. 每个假设单独报告，包含统计量和效应量
3. 使用表格和图表辅助展示
4. 结果描述客观，不做过度解读
5. 报告完整的统计信息（统计量、自由度、p值、效应量）
6. 字数建议：3000-5000字
