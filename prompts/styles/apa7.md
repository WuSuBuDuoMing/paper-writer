# APA 7th Edition Style Guide

## 论文结构

```
Title Page
Abstract
1. Introduction
2. Literature Review
3. Method
4. Results
5. Discussion
6. Conclusion
References
Appendices
```

## 格式要求

### 页面设置
- 页边距：1英寸（2.54cm）各边
- 字体：12pt Times New Roman
- 行距：双倍行距
- 段落首行缩进：0.5英寸（1.27cm）

### 标题层级
```
Centered, Bold, Title Case          → Level 1
Flush Left, Bold, Title Case        → Level 2
Flush Left, Bold Italic, Title Case → Level 3
Indented, Bold, Title Case, Period  → Level 4
Indented, Bold Italic, Title Case, Period → Level 5
```

## 文内引用格式

### 单作者
```
Wang (2024) 指出...
（Wang, 2024）
```

### 双作者
```
Wang and Li (2024) 发现...
（Wang & Li, 2024）
```

### 三及以上作者
```
首次：Wang et al. (2024)
后续：Wang et al. (2024)
```

### 直接引用
```
Wang (2024, p. 15) 指出："原文内容"。
```

## 参考文献格式

### 期刊文章
```
Author, A. A., Author, B. B., & Author, C. C. (2024). Title of article in sentence case. Title of Periodical in Title Case, volume(issue), pages. https://doi.org/10.xxxx/xxxxx
```

### 书籍
```
Author, A. A. (2024). Title of work in sentence case (Edition). Publisher.
```

### 会议论文
```
Author, A. A., & Author, B. B. (2024, March 15-18). Title of paper in sentence case. Conference Name, Location. Publisher. https://doi.org/xxxxx
```

### 学位论文
```
Author, A. A. (2024). Title of dissertation in sentence case [Doctoral dissertation, University Name]. Database Name. https://xxxxx
```

### 网页
```
Author, A. A. or Group Name. (Year, Month Day). Title of page in sentence case. Site Name. https://xxxxx
```
