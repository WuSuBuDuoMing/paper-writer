# Results Prompt

## 使用方法

将此 Prompt 发送给 Paper Writer Agent，替换 `{变量}` 为你的实际内容。

---

请为以下论文撰写结果/数据分析章节：

**研究假设/问题**：{列出所有假设或研究问题}
**分析方法**：{使用的统计分析方法}
**数据结果**：{粘贴统计分析输出结果}
**显著性水平**：{如 p < 0.05}
**报告格式**：{APA 7th / IEEE / GB/T 7714}
**目标字数**：{字数要求，默认 3000-5000 字}

### 输出结构

```
4.1 描述性统计
    4.1.1 样本特征
    4.1.2 变量描述性统计
    4.1.3 相关性分析

4.2 信效度检验
    4.2.1 信度分析
    4.2.2 效度分析

4.3 假设检验
    4.3.1 [假设1] 的检验结果
    4.3.2 [假设2] 的检验结果
    ...

4.4 稳健性检验
    4.4.1 替代模型验证
    4.4.2 内生性处理（如适用）

4.5 结果汇总
    - 表格汇总所有假设检验结果
```

### 写作规范

1. 先描述样本特征，再呈现分析结果
2. 每个假设单独报告，包含统计量和效应量
3. 使用表格和图表辅助展示
4. 结果描述客观，不做过度解读
5. 报告完整的统计信息（统计量、自由度、p值、效应量）
6. 字数建议：3000-5000字
7. 三线表格式，表题在上，注释在下

### 示例输入

```
研究假设：
H1: AI 辅助教学显著提升学生编程成绩
H2: AI 辅助教学提升学生学习动机
H3: 学习动机在 AI 辅助教学与编程成绩间起中介作用

分析方法：独立样本 t 检验、多元回归、Bootstrap 中介效应检验（SPSS + PROCESS）

数据结果：
- 实验组编程成绩 M=82.3, SD=8.5 (N=120)
- 对照组编程成绩 M=71.5, SD=10.2 (N=118)
- t(236)=4.52, p<0.001, Cohen's d=0.59
- 中介效应 Bootstrap 95% CI=[0.12, 0.38]，不包含零

显著性水平：p < 0.05
报告格式：APA 7th
```
