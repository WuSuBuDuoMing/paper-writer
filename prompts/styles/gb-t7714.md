# GB/T 7714-2015 Style Guide

## 论文结构（中文学术论文）

```
摘要
关键词
1 引言
2 文献综述
3 研究方法
4 结果与分析
5 讨论
6 结论
参考文献
致谢
附录
```

**学位论文结构**（硕博论文）：
```
独创性声明和使用授权书
中文摘要
英文摘要
目录
图表清单（可选）
1 引言
  1.1 研究背景
  1.2 研究目的与意义
  1.3 国内外研究现状
  1.4 研究内容与方法
  1.5 论文结构安排
2~N 各章节
结论与展望
参考文献
致谢
附录
攻读学位期间发表的论文
```

## 格式要求

### 页面设置
- 页边距：上下2.54cm，左右3.17cm
- 字体：正文宋体小四号，标题黑体
- 行距：1.5倍行距
- 段落首行缩进：2字符

### 标题层级
```
1 一级标题（黑体，三号，居中）
1.1 二级标题（黑体，四号，顶格）
1.1.1 三级标题（黑体，小四号，顶格）
```

### 摘要格式
```
摘  要（黑体，三号，居中）

[摘要正文，宋体小四号，300-500字]

关键词：关键词1；关键词2；关键词3（3-8个）
```

### 英文摘要格式
```
Abstract（Times New Roman，三号，居中）

[English abstract, Times New Roman, 小四号]

Keywords: keyword1; keyword2; keyword3
```

## 文内引用格式

### 顺序编码制（期刊论文常用）
```
...如文献[1]所述...
...多项研究[1-3]表明...
...根据文献[1,3,5]...
...Wang等[2]提出的框架...
```

### 著者-出版年制（学位论文常用）
```
...如王某某(2024)所述...
...Wang等(2024)的研究表明...
...研究表明(李四, 2023)...
```

## 参考文献格式

### 期刊文章 [J]
```
[序号] 作者. 题名[J]. 刊名, 出版年, 卷(期): 起止页码.
```

**示例**：
```
[1] 张三, 李四. 人工智能在教育中的应用研究[J]. 教育研究, 2024, 45(3): 12-20.
[2] CHEN W, ZHANG L. Deep learning for medical image analysis[J]. Nature Medicine, 2023, 29(5): 1123-1135.
```

### 书籍 [M]
```
[序号] 作者. 书名[M]. 版次. 出版地: 出版者, 出版年: 起止页码.
```

**示例**：
```
[3] 王五. 机器学习导论[M]. 3版. 北京: 清华大学出版社, 2024: 56-78.
[4] GOODFELLOW I, BENGIO Y, COURVILLE A. Deep Learning[M]. Cambridge: MIT Press, 2016: 98-156.
```

### 学位论文 [D]
```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.
```

**示例**：
```
[5] 李某某. 深度学习在自然语言处理中的应用研究[D]. 北京: 北京大学, 2024.
[6] ZHANG S. Federated learning for privacy-preserving AI[D]. Stanford: Stanford University, 2023.
```

### 会议论文 [C]
```
[序号] 作者. 题名[C]//会议录名. 出版地: 出版者, 出版年: 起止页码.
```

**示例**：
```
[7] VASWANI A, SHAZEER N, PARMAR N, 等. Attention is all you need[C]//Proc NeurIPS. Long Beach: Curran Associates, 2017: 5998-6008.
```

### 专利 [P]
```
[序号] 发明人. 专利题名: 专利号[P]. 公开日期.
```

**示例**：
```
[8] 王某某. 一种基于深度学习的图像识别方法: CN202410123456.7[P]. 2024-06-15.
```

### 标准 [S]
```
[序号] 标准编号, 标准名称[S]. 出版地: 出版者, 出版年.
```

**示例**：
```
[9] GB/T 7714-2015, 信息与文献 参考文献著录规则[S]. 北京: 中国标准出版社, 2015.
```

### 电子文献 [EB/OL]
```
[序号] 作者. 题名[EB/OL]. (发表日期)[引用日期]. 网址.
```

**示例**：
```
[10] OpenAI. GPT-4 Technical Report[EB/OL]. (2023-03-15)[2024-06-01]. https://openai.com/research/gpt-4.
[11] 国务院. 新一代人工智能发展规划[EB/OL]. (2017-07-08)[2024-01-15]. https://www.gov.cn/zhengce/2017-07/08/content_5239786.htm.
```

### 报纸 [N]
```
[序号] 作者. 题名[N]. 报纸名, 出版日期(版次).
```

### 报告 [R]
```
[序号] 作者. 题名[R]. 出版地: 出版者, 出版年.
```

## 作者规则

| 作者数量 | 中文文献 | 外文文献 |
|---------|---------|---------|
| 1-3人 | 全部列出，逗号分隔 | 全部列出 |
| 3人以上 | 前3人后加"等" | 前3人后加"et al." |
| 外文作者 | 姓在前名在后，名缩写 | 同左 |

**示例**：
```
- 1-3人：张三, 李四, 王五.
- 3人以上：张三, 李四, 王五, 等.
- 外文：Vaswani A, Shazeer N, Parmar N, et al.
```

## 中英文文献混合著录规则

- 中文文献排在前面，英文文献排在后面（顺序编码制可按引用顺序）
- 同一语种内部按作者姓氏字母/拼音排序（著者-出版年制）
- 外文期刊名使用缩写（参照 ISO 4 标准）
- 中文文献的英文翻译括注在中文题名后（可选）
