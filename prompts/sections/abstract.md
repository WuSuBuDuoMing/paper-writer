# Abstract Prompt

## 使用方法

将此 Prompt 发送给 Paper Writer Agent，替换 `{变量}` 为你的实际内容。

---

请为以下论文撰写中英文摘要：

**研究主题**：{研究主题}
**研究问题**：{研究问题}
**研究方法**：{研究方法}
**主要发现**：{主要发现}
**结论**：{结论}
**目标期刊**：{目标期刊名称（可选）}
**语言要求**：{中文/英文/双语}

### 输出要求

**中文摘要**（300-500字）：
- 背景与目的（1-2句）
- 方法（1-2句）
- 结果（2-3句）
- 结论（1-2句）
- 关键词（3-5个）

**English Abstract** (150-300 words):
- Background & Objective (1-2 sentences)
- Methods (1-2 sentences)
- Results (2-3 sentences)
- Conclusion (1-2 sentences)
- Keywords (3-5)

### 摘要写作规范

1. 使用第三人称或无人称句式
2. 避免引用参考文献
3. 避免使用缩写（首次出现时）
4. 结果部分应包含具体数据
5. 语言简洁，每句话信息密度高
6. 中英文摘要内容应对应但非逐字翻译

### 示例输入

```
研究主题：大语言模型在高校编程教学中的应用效果研究
研究问题：AI 辅助编程教学是否显著提升学生的编程能力？
研究方法：准实验设计，实验组（AI 辅助教学，N=120）vs 对照组（传统教学，N=118），前后测设计
主要发现：实验组编程测试成绩显著优于对照组（M=82.3 vs M=71.5, t(236)=4.52, p<0.001, d=0.59）
结论：大语言模型辅助编程教学能有效提升学生的编程能力和学习动机
```
