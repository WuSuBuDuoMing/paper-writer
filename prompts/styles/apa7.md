# APA 7th Edition Style Guide

## 论文结构

```
Title Page
Abstract
1. Introduction
2. Literature Review
3. Method
4. Results
5. Discussion
6. Conclusion
References
Appendices
```

## 格式要求

### 页面设置
- 页边距：1英寸（2.54cm）各边
- 字体：12pt Times New Roman
- 行距：双倍行距
- 段落首行缩进：0.5英寸（1.27cm）

### 标题层级
```
Centered, Bold, Title Case          → Level 1
Flush Left, Bold, Title Case        → Level 2
Flush Left, Bold Italic, Title Case → Level 3
Indented, Bold, Title Case, Period  → Level 4
Indented, Bold Italic, Title Case, Period → Level 5
```

### Title Page 格式
```
                        Title of the Paper
                          Author Name
                    Department, University
                       Course: Course Name
                       Instructor: Dr. Name
                           Date
```

- 研究论文（Student Paper）：标题、作者、学校、课程、教师、日期
- 专业论文（Professional Paper）：标题、作者、作者单位、作者注释

## 文内引用格式

### 单作者
```
Wang (2024) 指出...
（Wang, 2024）
```

### 双作者
```
Wang and Li (2024) 发现...
（Wang & Li, 2024）
```

### 三及以上作者
```
首次：Wang et al. (2024)
后续：Wang et al. (2024)
```

### 直接引用
```
Wang (2024, p. 15) 指出："原文内容"。
```

### 多篇文献同时引用
```
多项研究（Li, 2023; Wang, 2024; Zhang et al., 2022）表明...
```

### 机构作者
```
World Health Organization (WHO, 2023) 报告...
首次引用后：WHO (2023)
```

### 二手文献
```
Smith (as cited in Wang, 2024) 提出...
```

## 参考文献格式

### 期刊文章
```
Author, A. A., Author, B. B., & Author, C. C. (2024). Title of article in sentence case. Title of Periodical in Title Case, volume(issue), pages. https://doi.org/10.xxxx/xxxxx
```

**示例**：
```
Wang, M., & Li, H. (2024). The impact of AI on higher education. Journal of Educational Technology, 45(3), 112-128. https://doi.org/10.1000/example2024
```

### 书籍
```
Author, A. A. (2024). Title of work in sentence case (Edition). Publisher.
```

**示例**：
```
Creswell, J. W., & Creswell, J. D. (2018). Research design: Qualitative, quantitative, and mixed methods approaches (5th ed.). Sage.
```

### 会议论文
```
Author, A. A., & Author, B. B. (2024, March 15-18). Title of paper in sentence case. Conference Name, Location. Publisher. https://doi.org/xxxxx
```

**示例**：
```
Vaswani, A., Shazeer, N., Parmar, N., Uszkoreit, J., Jones, L., Gomez, A. N., Kaiser, L., & Polosukhin, I. (2017). Attention is all you need. Advances in Neural Information Processing Systems, 30.
```

### 学位论文
```
Author, A. A. (2024). Title of dissertation in sentence case [Doctoral dissertation, University Name]. Database Name. https://xxxxx
```

**示例**：
```
Zhang, L. (2023). AI-assisted learning in Chinese universities: A mixed-methods study [Doctoral dissertation, Peking University]. CNKI. https://www.cnki.net/xxxxx
```

### 网页
```
Author, A. A. or Group Name. (Year, Month Day). Title of page in sentence case. Site Name. https://xxxxx
```

**示例**：
```
OpenAI. (2023, March 14). GPT-4 technical report. https://openai.com/research/gpt-4
```

### 报告
```
Author, A. A. (Year). Title of report in sentence case (Report No. XXX). Publisher. https://xxxxx
```

### 编辑书籍中的章节
```
Author, A. A. (2024). Title of chapter in sentence case. In E. E. Editor (Ed.), Title of book in sentence case (pp. xx-xx). Publisher.
```

## 统计结果报告规范

| 统计方法 | 报告格式 |
|---------|---------|
| t 检验 | t(df) = x.xx, p = .xxx, d = x.xx |
| ANOVA | F(df1, df2) = x.xx, p = .xxx, eta^2 = x.xx |
| 相关 | r(df) = .xx, p = .xxx |
| 回归 | beta = .xx, SE = .xx, t(df) = x.xx, p = .xxx |
| 卡方 | chi^2(df, N = xxx) = x.xx, p = .xxx |

### 规则
- p 值：小数点前的 0 省略（p = .03 而非 p = 0.03），p < .001 保留
- 统计量：保留两位小数
- 效应量：保留两位小数
- 均值和标准差：M = xx.xx, SD = xx.xx

## 表格与图片

### 表格
- 三线表（顶线、栏目线、底线，无竖线）
- 表号和表题在表格上方，左对齐，斜体
- 注释在表格下方
- 表格上方的引用文字应说明表的内容

### 图片
- 图号和图题在图片下方，左对齐，斜体
- 分辨率 >= 300 DPI
- 色盲友好配色方案
