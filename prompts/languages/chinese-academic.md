# Chinese Academic Writing Guide

## 语言规范

### 称谓用语
| 使用 | 避免 |
|------|------|
| 本文、本研究、本论文 | 我们、笔者、作者 |
| 该方法、该技术 | 这个方法、这个技术 |
| 研究表明、结果显示 | 我发现、我觉得 |
| 具有重要意义 | 很重要 |
| 显著影响 | 影响很大 |

### 学术连接词

#### 因果关系
因此、由此可见、综上所述、由此可得、这说明、这意味着

#### 转折关系
然而、但是、尽管如此、不过、与之相反、相比之下

#### 递进关系
此外、另外、同时、不仅如此、进一步地、更重要的是

#### 并列关系
首先...其次...最后、一方面...另一方面、不仅...而且

#### 举例说明
例如、比如、具体而言、以...为例、如表X所示

#### 总结归纳
总之、简而言之、概括来说、归纳起来

## 常见问题

### 口语化→学术化
| 口语 | 学术 |
|------|------|
| 很多 | 大量、众多、显著 |
| 证明了 | 验证了、证实了、表明 |
| 但是 | 然而、尽管如此 |
| 所以 | 因此、由此可见 |
| 重要的 | 关键的、核心的、显著的 |
| 做了 | 实施了、开展了、进行了 |
| 看出来 | 可以发现、结果显示 |
| 搞清楚 | 阐明、厘清、明确 |

### 段落结构
每段应包含：
1. **主题句**：概括本段核心观点
2. **论据**：数据、引文、例证
3. **分析**：对论据的解读
4. **小结/过渡**：总结或引出下一段

### 标点符号
- 中文使用全角标点：，。；：！？
- 英文和数字使用半角标点
- 中英文之间加空格：中文 English 中文
- 引号使用「」或 ""
- 破折号使用——（两个减号）
- 省略号使用……（两个省略号）
