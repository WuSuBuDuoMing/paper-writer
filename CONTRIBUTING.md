# Contributing to Paper Writer

感谢你对 Paper Writer 的兴趣！欢迎贡献代码和想法。

Thank you for your interest in Paper Writer! Contributions are welcome.

---

## 🚀 Getting Started

1. **Fork** the repository
2. **Clone** your fork
   ```bash
   git clone https://github.com/your-username/paper-writer.git
   ```
3. **Create a branch** for your changes
   ```bash
   git checkout -b feature/your-feature-name
   ```

---

## 📋 Ways to Contribute

### Adding a New Agent
1. Create a new `.md` file in `.claude/agents/`
2. Follow the existing agent format (name, capabilities, workflow)
3. Update README.md to include the new agent

### Adding a Prompt Template
1. Create a new `.md` file in `prompts/sections/`
2. Include: variable placeholders, output structure, writing guidelines
3. Update README.md

### Adding a Formatting Style
1. Create a new `.md` file in `prompts/styles/`
2. Cover: page setup, heading levels, in-text citation format, reference format
3. Include examples for each reference type

### Adding a Language Guide
1. Create a new `.md` file in `prompts/languages/`
2. Cover: academic vocabulary, common mistakes, paragraph structure
3. Include before/after examples

### Bug Fixes & Improvements
- Fix typos or errors in existing files
- Improve prompt templates
- Add more examples

---

## 🎨 Conventions

- All files should be in **Markdown** format
- Use **bilingual** content (English + Chinese) where appropriate
- Include **examples** for all formatting rules
- Use `[PLACEHOLDER]` syntax for variable placeholders in prompts
