# Introduction Prompt

## 使用方法

将此 Prompt 发送给 Paper Writer Agent，替换 `{变量}` 为你的实际内容。

---

请为以下论文撰写第一章：绪论/Introduction。

**研究主题**：{研究主题}
**学科领域**：{学科领域}
**研究背景**：{研究背景概述}
**拟解决的问题**：{核心研究问题}
**预期贡献**：{预期理论/实践贡献}

### 输出结构

```
1.1 研究背景
    - 宏观背景（社会/行业/技术发展趋势）
    - 微观背景（具体问题/现象）
    - 研究的必要性和紧迫性

1.2 研究问题与目标
    - 核心研究问题（1个）
    - 子研究问题（2-4个）
    - 研究目标

1.3 研究意义
    - 理论意义
    - 实践意义

1.4 研究内容与方法
    - 研究内容概述
    - 研究方法概述
    - 技术路线（可选）

1.5 论文结构安排
    - 各章节内容简介
```

### 写作规范

1. 研究背景从宏观到微观，逐步聚焦
2. 研究问题要具体、明确、可操作
3. 研究意义要与研究问题紧密对应
4. 使用"漏斗式"结构：背景→问题→意义→方法
5. 字数建议：3000-5000字
